# ROSEBUD
This github repository contains all images and masks pertaining to the River Obstacle Segmentation 
Enroute By USV Dataset (ROSEBUD) Fluvial Dataset. The dataset contains annotated images for training
supervised networks on the task of recognizing and segmenting fluvial scnenes. In total there are 550 
images split across two sub-sets of data detailed below.
